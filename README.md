# 🎵 Music Player Using Data Structures

A console-based Music Player application developed in **C++** as part of the **Data Structures and Algorithms** course.

## Features
- Create a playlist
- Add new songs
- Delete songs by search or position
- Display the playlist
- Search for a song
- Play songs
- View recently played songs
- Display the last played song
- Count total songs
- Sort the playlist
- Save and load playlists using file handling
  
## Data Structures Used
- Doubly Linked List
- Stack
- File Handling

## Technologies
- C++
- Visual Studio Code
